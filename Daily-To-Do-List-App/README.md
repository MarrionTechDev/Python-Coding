# 📝 Daily To-Do List App (CLI)

A command-line-based To-Do List application built in Python.

This project started as a simple learning exercise and is being continuously improved as I learn more about clean code, input validation, program structure, and version control.

The long-term goal is to evolve this into a fully functional standalone productivity application (CLI → GUI → possibly web-based).

---

## 🚀 Features (Current Version)

- Add tasks
- View tasks
- Remove tasks
- Input validation for safer user interaction
- Modular function-based structure

---

## 🛠 Built With

- Python 3
- Basic control flow (loops, conditionals)
- Functions
- Input validation
- Version control with Git

## ▶ How to Run

Clone the repository:

```bash
git clone https://github.com/yourusername/Daily-To-Do-List-App.git
cd Daily-To-Do-List-App
```

Run the application:

```bash
python daily-to-do-list.py
```
