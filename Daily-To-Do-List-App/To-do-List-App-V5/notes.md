# 📝 Daily To-Do List App (With GUI)

A To-Do List application with TKinter GUI built in Python.

This project started as a simple learning exercise and is being continuously improved as I learn more about clean code, input validation, program structure, and version control.

The long-term goal is to evolve this into a fully functional standalone productivity application (CLI → GUI → possibly web-based).

Finally moved away from CLI 
---

## 🚀 Features (Current Version)

- Add task with button and Return Key
- View tasks through Listbox 
- Remove tasks using the Delete button
- Task input using an entry box
- Notebook section using Textbox
- Date and number of task completed displayed at the top of the window
- Modular Class-based structure
- Save tasks to file (JSON persistence)
- Edit existing tasks
- Double click to mark task complete
- Random Task button to choose a random task
