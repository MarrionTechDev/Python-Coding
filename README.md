# python-mini-projects
This is a collection of mini projects I did to get myself familiar with coding in Python, learning coding concepts, and problem-solving.

## Projects Included

### 🎲 Dice Roller
Simulates rolling a six-sided dice using Python's random module.

### 🔢 Number Guessing Game
User guesses a randomly generated number with hints (too high / too low).

### ✊ Rock Paper Scissors
Classic game against the computer.

## Concepts Practiced
- Loops
- Conditionals
- Functions
- User Input
- Python random module
- File handling

## How to Run
Run any file using:

python filename.py
