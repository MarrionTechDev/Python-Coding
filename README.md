# Python Mini Projects

This repository contains a collection of beginner-to-intermediate Python projects created to strengthen my understanding of programming fundamentals, problem-solving, and software structure.

These projects were inspired by CodeWithMoshe's "Python Projects for Beginners" guide, with additional modifications and feature enhancements implemented independently.

---

## Projects Included

### 🎲 Dice Roller
A dice rolling simulator that:
- Supports rolling multiple dice
- Tracks total rolls per session
- Allows repeated gameplay

**Concepts:** random module, loops, session state tracking, user input

---

### 🔢 Number Guesser
A number guessing game where:
- The user can define the guessing range
- Guess attempts are limited
- The correct number is revealed if attempts run out
- The program tracks the best score (fewest attempts)

**Concepts:** conditionals, loops, random number generation, state tracking

---

### ✊ Rock Paper Scissors
A terminal-based Rock Paper Scissors game featuring:
- Emoji-based output
- Best 2 out of 3 win system
- Win/Loss/Tie tracking
- Two-player mode

**Concepts:** game logic, input validation, score tracking, functions
